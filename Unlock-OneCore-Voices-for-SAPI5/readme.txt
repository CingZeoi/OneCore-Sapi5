How to Unlock Mobile Text-To-Speech Voice in Windows 10


By default, the Microsoft mobile voice is locked for using in
text-to-speech software via SAPI 5. You can unlock it with a
simple registry tweak. Download the archive, extract the file
for your language and for your version of the operating system
("mobile_x86.reg" for 32-bit and "mobile_x64.reg" for 64-bit),
click the right mouse button on the file's name and choose the
context menu item "Merge". The Microsoft mobile voice will
appear in the list of the available voices in Balabolka.


Attention! The changes in the system registry do not add files
of voices to the computer's disk. To add the voice for the new
language, you need to take the following steps:

1. Navigate to "Settings" (click the "Settings" icon on the
   start menu of Windows).
2. Click "Time & Language".
3. Select "Region & Language".
4. Select a new region from the menu.
5. Click "Add a language".
6. Select the appropriate language.
7. Select the local dialect for the language.
8. Click the language you just added.
9. Click "Options".
10. Click the "Download" buttons under "Download language pack"
    and "Speech". Wait for both to install. It could take a few
    minutes and you may have to click the button again to restart
    one of them if it gives you a message that it failed.
11. Restart your computer.

###

Par défaut, la voix Microsoft mobile est bloquée pour
l'utilisation dans les logiciels de lecture à haute voix via
SAPI 5. Vous pouvez la débloquer par une simple modification
du registre. Téléchargez l'archive, extrayez le fichier pour
votre langue et pour votre version du système d'exploitation
(« mobile_x86.reg » pour 32bit et « mobile_x64.reg » pour 64bit),
cliquez avec le bouton droit de la souris sur le nom du fichier
et choisissez l'élément de menu contextuel « Fusionner ».
La voix Microsoft mobile apparaîtra dans la liste des voix
disponibles dans Balabolka.

###

Standardmäßig ist die Microsoft-Mobil-Stimme für die Verwendung
in Text-to-Speech-Software mittels SAPI 5 gesperrt. Sie können
die Stimme aber mit einem einfachen Registrierungstrick
freischalten. Laden Sie das Archiv, entpacken Sie die Datei in
Ihrer Sprache und für Ihre Version des Betriebssystems
("mobile_x86.reg" für 32-bit und "mobile_x64.reg" für 64-bit),
klicken Sie mit der rechten Maustaste auf den Namen der Datei
und wählen Sie im Kontextmenü den Punkt "Zusammenführen".
Die Microsoft-Mobil-Stimme wird in der Liste der verfügbaren
Stimmen in Balabolka erscheinen.

###

Standaard zijn de mobiele stemmen geblockeerd om gebruikt te
worden door spraaksoftware die werkt via SAPI 5. Je kan dit
eenvoudig de-blockeren middels een registry-aanpassing.
Download het archiefbestand en pak het uit.
Kies je taal en het bestand voor je Windows versie
("mobile_x86.reg" voor 32bit en "mobile_x64.reg" voor 64bit),
Klik er met rechts op en kies in het contekstmenu voor "Samenvoegen".
De Microsoft mobile stem zal nu beschikbaar worden in het overzicht
van beschikbare stemmen in Balabolka.

###

По подразбиране гласовете от групата Microsoft Mobile са
заключени за използване само в определен тип приложения за изход
чрез синтезиран говор (например вграденият екранен четец
"Разказвач"). Можете да ги отключите чрез проста промяна в
регистъра на Windows. Изтеглете архива. Извлечете файла за вашия
език и за вашата архитектура на операционната система
("mobile_x86.reg" за 32 бита и "mobile_x64.reg" за 64 бита).
Щракнете с десния бутон на мишката върху името на файла (за да
извикате контекстното меню за него) и от менюто изберете елемента
"Merge" (алтернативно, просто натиснете ENTER или щракнете
двукратно с левия бутон на мишката върху файла). Задействайте
бутона "Да" в диалоговите прозорци за контрол на потребителските
акаунти (UAC) ако се появи такъв и в питащия ви дали сте сигурни
че желаете да добавите тези ключове към регистъра на Windows.
Съответните гласове от групата Microsoft Mobile ще се появят в
списъка с наличните гласове в програми като Balabolka.

###

Голоса Microsoft Mobile предназначены для использования
в приложениях из магазина Windows Store. По умолчанию к голосам
нельзя обратиться при помощи функций SAPI 5. Однако, есть
простой способ решить данную проблему. Скачайте архив, извлеките
из него файл для соответствующего языка и соответствующей версии
операционной системы ("mobile_x86.reg" для 32-битной версии и
"mobile_x64.reg" для 64-битной версии), кликните правой кнопкой
мыши на имени файла и выберите пункт "Слияние" в контекстном
меню. Данные о голосе добавятся в системный реестр Windows, и
голос можно будет использовать в программе "Балаболка".
